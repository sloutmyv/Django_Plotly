Hello World !


Ce site internet a entièrement été construit dans l’optique de participer au “Pacifique Dataviz Challenge 2023”.


Le challenge en quelques mots


L’objectif du Challenge est de proposer un ensemble de visuels permettant de simplifier la compréhension et l'interprétation de multiples grandeurs récoltées dans d’imposantes bases de données.

Ces visuels seront jugés sur une grille de 4 critères :
  • le storytelling ou la capacité à raconter une histoire
  • le design
  • l’innovation, la créativité et l’originalité
  • la solution technique et la profondeur analytique
Les visuels peuvent être abordés en deux catégories : les visuels statiques et/ou les visuels interactifs.

Le thème 2023 : l’alimentation

Pour plus de détails : dataviz.pacificdata.org



Les bases de données   

  • Pacific food and beverage trade : The Pacific Food Trade Database
  • Food security : Nutritional facts by type of food, by geography, sex, age and urbanization (Kiribati, Solomon Islands and Vanuatu)
  • Fruits and vegetables market in New Caledonia
  • Food prices in New Caledonia



Les outils   

Les bases de données ont été exportées depuis les répertoires internet en fichier “.csv”.

L’analyse a été réalisée en langage python avec des librairies dédiées à la data science et à la visualisation graphique telles que pandas, numpy, matplotlib, seaborn et bien d’autres…
L’ensemble de notebook d’analyse sont disponibles sur un répertoire github : ici

Le site internet a été construit en python avec le framework Django. Les bases de données du site sont gérées avec PostgreSQL. Le site est hébergé par Heroku et AWS (pour les statics et les médias).
L’ensemble de la structure du site est disponible sur un répertoire github : ici



Présentation du site / Philosophie de l’analyse   

Chaque base de données est analysée dans une page dédiée :

  • Food Trade : ici
  • Food Security : ici
  • Fruits & Légumes en Nouvelle-Calédonie : ici
  • Prix de l'alimentation en Nouvelle-Calédonie : ici
Chaque page est construite de la même façon. En préambule, une description de la base de donnée brute fournie puis une série de plusieurs analyses thématiques dans lequelles sont présentées un ou plusieurs visuels.

La plupart des visuels sont associés à un texte explicatif simple donnant quelques pistes pour entamer la lecture et l'analyse.
Certains visuels ont été agrémentés avec des informations ou des jeux de données externes. Dans ce cas, les références sont citées.

Une page est dédiée aux éléments de contexte pour introduire les enjeux du thème et permettre au lecteur de mettre en perspective les informations contenues dans les visuels.
